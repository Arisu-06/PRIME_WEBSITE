# PRIME_WEBSITE

Одностраничный сайт на тему **«Цифровой портрет профессии: инженер по эксплуатации сети»**.

## Что есть на сайте

* описание профессии простыми словами;
* 5–7 рабочих задач инженера;
* hard skills и soft skills;
* путь старта для студента;
* таймлайн «Один день из жизни»;
* блок о важности профессии для ЭР-Телеком;
* анимированный фон;
* интерактивный мини-тест.

## Технологии

* HTML
* CSS
* JavaScript
* GitHub
* Vercel

## Использование ИИ

ИИ использовался для создания структуры сайта, текстов, дизайна, интерактива и исправления ошибок.

## Промт

Создай современный одностраничный сайт на HTML, CSS и JavaScript на тему «Цифровой портрет профессии: инженер по эксплуатации сети».

Сайт должен простыми словами объяснять профессию, быть визуально современным, технологичным и аккуратным.

Добавь главный экран, описание профессии, 5–7 задач, hard skills и soft skills, карьерный путь студента, один день из жизни, важность роли для и интерактив. Сделай анимировнные кнопки с переливанием и подсветкой через наведение. 

Оформление: светлый фон, градиентные акценты, карточки, плавные анимации, эффекты, красивые кнопки и адаптивность.

Код должен быть готовым в файлах: `index.html`, `style.css`, `script.js`.

Далее мы вносили правки и дорабатывали основной промт!
